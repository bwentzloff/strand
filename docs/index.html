Strand Interpreter

Turn DNA sequences into functional code

About Strand

The Strand Interpreter is an esoteric programming language designed to execute DNA sequences as code. Inspired by genetic programming, Strand can interpret DNA sequences or FASTA files, producing synthesized proteins, numerical results, and visual pathway graphs.

Features

  • DNA sequence execution with start (ATG) and stop (TAA, TAG, TGA) codons.
  • Supports FASTA file inputs.
  • Textual logs summarizing synthesized proteins.
  • Numerical results for arithmetic pathways.
  • Visual graphs rendering pathway interactions (planned).
  • Extensible "virus library" system for custom functions.

Installation

To install Strand, follow these steps:

# Clone the repository
git clone <repository_url>
cd strand_interpreter

# Compile the interpreter
pip install pyinstaller
pyinstaller --name strand --onefile interpreter.py

# Move the executable to a directory in your PATH
sudo mv dist/strand /usr/local/bin/

Usage

Direct DNA Input

Run a DNA sequence directly:

strand ATGACCGATTAA

FASTA File Input

Run a DNA sequence from a FASTA file:

strand -f example.fasta

Example Programs

Basic Protein Synthesis

Input:
ATG ACC GAT TAA

Output:
Results:
Proteins: [['Protein_X', 'Transport']]
Environment: {'TEMP': 'NEUTRAL', 'ENERGY': 'MEDIUM'}

Using Viruses

Input:
TAA GGA AAGGTTCCCGAT ATG ACC GAT TAA

Output:
Results:
Proteins: [['Protein_X', 'Transport', 'Protein_Sum']]
Environment: {'TEMP': 'NEUTRAL', 'ENERGY': 'MEDIUM'}
