Strand Language Documentation

About Strand

Strand is an esoteric programming language inspired by biological DNA sequences. It interprets DNA-like input to produce complex, dynamically changing outputs while simulating chemical pathways and interactions.

Features

  • DNA-inspired syntax using the ACGT alphabet
  • Dynamic pathways and proteins for advanced logic
  • Multi-environment support with threaded execution
  • Relevant JSON output for tick-based simulations

JSON Output for Relevant Ticks

The Strand interpreter outputs JSON data that only logs ticks where meaningful changes occur, ensuring compact and relevant output for easier analysis.

Example DNA Input

TGC TAA AAC AAG AAC AAG AAC ACA AAG AAG AAC AAA GGA  # FOREST: TEMP=HIGH, ENERGY=HIGH
TGC TAA AAC AAC AAC AAC AAC ACC ACG AAT GGG          # DESERT: TEMP=MEDIUM, ENERGY=LOW
ATG ACC GAT CCG TGG TAA                              # Shared pathway
TGC TAA ACC GAT                                      # Remove edge ACC → GAT in FOREST
TGC CAG GAT                                          # Duplicate GAT node in FOREST
TGC TGC TAA ACC GAT                                  # Insert TAA between ACC → GAT in DESERT
TGC GTT GAT CCG 3                                    # Deactivate GAT → CCG after 3 ticks in DESERT

How It Works

  1. Define environments using DNA-like syntax.
  2. Create pathways and apply modifications using Strand's codons.
  3. Run the interpreter to generate relevant JSON output.
  4. Visualize and analyze tick-based changes.

Base-6 Encoding for Strings

In Strand, strings such as environment names must be encoded using the ACGT alphabet. The encoding is done using base-6, where:

  • Each character in the string is converted to its ASCII value.
  • The ASCII value is then represented in base-6 (digits: 0, 1, 2, 3, 4, 5).
  • Each base-6 digit is mapped to a codon:
    • 0 → AAA
    • 1 → AAC
    • 2 → AAG
    • 3 → ACA
    • 4 → ACC
    • 5 → ACG

Encoding Example

For the string `FOREST`:

AAC AAG AAC AAG AAC ACA AAG AAG AAC AAC AAC AAC AAG AAG ACA AAG ACA AAA

Decoding Example

The DNA `AAC AAG AAC AAG AAC ACA` decodes to `FOREST`.
